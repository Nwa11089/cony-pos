🏪 Flujo Principal: Pedido en Caja

┌──────────────────────────────────────────────────────────────────┐ │ INICIO DEL DÍA │ │ Abrir Corte de Caja │ └──────────────────────────────────────────────────────────────────┘ │ ▼ ┌───────────────────────────────────┐ │ 1️⃣ CLIENTE LLEGA │ │ Mesero saluda y toma orden │ └───────────────────────────────────┘ │ ▼ ┌───────────────────────────────────┐ │ 2️⃣ SELECCIONAR PRODUCTOS │ │ En Comandero: click en categoría │ │ click en producto → se agrega │ │ Ajustar cantidad si es necesario │ └───────────────────────────────────┘ │ ▼ ┌───────────────────────────────────┐ │ 3️⃣ COBRAR / ENVIAR │ │ Elegir: "Cobrar" o "Enviar a │ │ cocina". Seleccionar método de │ │ pago (Efectivo, Tarjeta, etc.) │ └───────────────────────────────────┘ │ ▼ ┌───────────────────────────────────┐ │ 4️⃣ SE IMPRIME │ │ Ticket en caja (cliente) │ │ Comanda en cocina (cocineros) │ └───────────────────────────────────┘ │ ▼ ┌───────────────────────────────────┐ │ 5️⃣ COCINA PREPARA │ │ Cocina ve la comanda impresa │ │ Prepara el pedido │ │ Cambia estado a "Completado" │ └───────────────────────────────────┘ │ ▼ ┌───────────────────────────────────┐ │ 6️⃣ ENTREGAR AL CLIENTE │ │ Mesero lleva el pedido │ │ Cambia estado a "Entregado" │ └───────────────────────────────────┘ │ ▼ ┌───────────────────────────────────┐ │ 7️⃣ FIN DEL DÍA │ │ Cerrar Corte de Caja │ │ Sistema calcula totales │ │ Corte con efectivo + tarjeta │ └───────────────────────────────────┘
1
🔓 Abrir Corte El cajero/admin va a Dashboard → "Abrir Corte". Ingresa el fondo inicial en efectivo. Sin corte abierto no se pueden registrar pedidos.
2
📋 Tomar Pedido En el comandero, seleccionar categoría (Hot dogs, Hamburguesas, etc.). Click en cada producto para agregarlo a la cuenta. Se puede ajustar cantidad y agregar notas.
3
💳 Cobrar Click "Cobrar" o "Enviar a Cocina". Elegir método de pago. El sistema descuenta inventario automáticamente si el producto tiene receta (ingredientes definidos).
4
🧾 Imprimir El pedido se encola en la BD. El print-client en la Mac local lo toma y lo imprime en la impresora correspondiente (caja o cocina según el rol).

🔄 Flujo de Roles

RolPuede hacerNo puede hacer
Superuser Todo: crear/eliminar usuarios, editar inventario, ver todo Nada. Es el nivel máximo.
Admin Gestión de productos, inventario, usuarios, cortes, delivery No puede eliminar usuarios superuser
User Tomar pedidos, ver dashboard, ver delivery No puede crear/editar productos ni usuarios
Repartidor Ver pedidos asignados, cambiar estado a entregado No puede tomar pedidos ni ver administración
